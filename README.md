# Alkemy Challenge (Backend)

## Front end
- [Repo](https://github.com/alstroemerieae/alkemy-challenge)

## Technologies

- Node.js
- Express
- Sequelize
- PostgreSQL

## Requirements
- [Node](https://nodejs.org/en/download/)
- [PostgreSQL](https://www.postgresql.org/download/)

## Install

```
git clone https://github.com/alstroemerieae/alkemy-challenge-backend.git

cd alkemy-challenge-backend

npm install

npm start
```
## API url
```
http://localhost:8080/api/operations
```
